Get Started

Optimized for Cloud Environment

QueryPie는 AWS, Microsoft Azure, Google Cloud Platform과 같은 클라우드 환경에 최적화되어 있습니다. 단 몇 초만에 여러 클라우드 및 리전에 흩어진 데이터베이스를 한 곳에서 관리할 수 있도록 동기화합니다.

One-Click Installation

Docker, Kubernetes를 비롯해 AWS(ECS, EKS), Google Cloud Platform(GKE), Microsoft Azure(AKS)와 같은 개발 환경에서는 복잡한 과정없이 단 몇분만에 간편하게 QueryPie를 설치할 수 있습니다.

All-In-One DB Solution

데이터베이스 접근제어, 멀티 클라우드, 멀티 리전 동기화, SQL 실행 및 DB 인증 기록 감사, 동적 / 정적 데이터 마스킹, Web SQL Editor
