# QueryPie ACP Community Edition

### 🧭 始める前に

**QueryPie Community Edition**は**QueryPie Enterprise**で提供される核心機能を体験できるようにします。

* Database Access Controller
* System Access Controller
* Kubernetes Access Controller
* Web Access Controller

ただし、Community Editionは**最大5名のユーザーアクティベーション**をサポートします。*（5名以上を登録する必要がある場合、*[*Standard Edition*](https://www.querypie.com/plans?acp)*または*[*Enterprise Edition*](https://www.querypie.com/plans?acp)*アップグレードを検討してください。）*

QueryPieは一般的なウェブアプリケーション形式で実行され、プロキシベースネットワークサーバ機能も含んでいます。

### 🐳 インストール方法

#### 1. ターミナルでコマンド実行

Linuxサーバのターミナルに接続した後、ホームディレクトリで次のコマンドを実行してください：
```
bash <(curl -s https://dl.querypie.com/setup.v2.sh)
```

⏱️ インストールは通常**7～10分**程度かかります。

#### 2. インストール中にライセンス発行を受ける

QueryPie Community Editionはライセンス登録が必要です。
インストールが進行中に[ライセンス申請書を提出](https://querypie.com/querypie/license/community/apply)すると、申請時に入力したメールアドレスに`.crt`テキストファイルが送信されます。

#### 3. インストール後接続

インストールが完了するとブラウザで次のアドレスに接続してください：
```
http://<server IP address>
or
https://<server IP address>
```

QueryPieがインストールされたLinuxサーバのIPアドレスはユーザーPCからアクセス可能である必要があります。

#### 4. ライセンス登録

メールで受け取った`.crt`ファイルをアップロードするか、PEM形式のテキスト内容をコピーして貼り付けてください。

<Callout type="info">
**ライセンス案内**
* QueryPieを使用するにはインストール後必ず有効なライセンスを登録する必要があります。
* ライセンスは`.crt`拡張子のテキストファイルで提供されます。
* ライセンスは申請時に入力したメールアドレスに送信されます。
* 各ライセンスは発行日から1年間有効です。
* ライセンスは申請者本人に発行され、第三者に譲渡できません。
</Callout>

基本アカウントでログインしてください。

新しいパスワードを設定した後サービスを使用できます。

* ID: `qp-admin`
* Password: `querypie`

<Callout type="info">
**QueryPie ACP Community Editionのインストール場所**

* Mac, Linux：インストールスクリプトを実行したフォルダ/querypie/&lt;バージョン&gt;
    * 例：/Users/brendon/querypie/11.5.0または/home/brendon/querypie/11.5.0（ユーザーホームディレクトリでインストールスクリプトが実行された場合）

**QueryPie ACP Community Editionの手動起動**

* インストール場所に移動します。
```
 cd ~/querypie/11.5.0
```

* 手動起動：database起動 → app起動の順序で起動します。
```
docker compose --profile=database up -d
docker compose --profile=app up -d
```

* 正常起動確認
```
docker compose --profile=app exec app readyz
```

正常に起動した場合、以下のような画面が出力されます。

<figure data-layout="center" data-align="center">
<img src="/installation/querypie-acp-community-edition/image-20260107-083457.png" alt="正常に起動した場合" width="736" />
<figcaption>
正常に起動した場合
</figcaption>
</figure>

**QueryPie ACP Community Editionの手動停止**

* インストール場所に移動します。
```
 cd ~/querypie/11.5.0
```

* 手動停止：app停止 → database停止の順序で停止します。
```
docker compose --profile=app down
docker compose --profile=database down
```

* 正常終了確認
```
docker compose ps
```
</Callout>

### 💬 サポートおよび問い合わせ

QueryPie Communityインストールおよび使用中に疑問点があれば[QueryPie AIP](https://app.querypie.com/)を通じて直接問い合わせできます。
以下の手順に従ってMCPで**QueryPie顧客センター**プリセットを設定すると、チャット画面で直接サポートを受けられます。

<figure data-layout="center" data-align="center">
<img src="/installation/querypie-acp-community-edition/image-20250718-070933.png" alt="image-20250718-070933.png" width="847" />
</figure>

1. Integrationタブ → Install QueryPie Customer Centerクリック。
2. Presetタブ → New Preset選択後、以下のように入力して`Save Changes`クリック：

* Name：`QueryPie`を入力（推奨）。
* MCP Servers & Tools：`Select All`を選択。

1. Chatタブ → `@QueryPie`選択後、質問入力。

また、[公式QueryPie Discordチャネル](https://discord.com/invite/Cu39M55gMk)に参加して他のユーザーと質問と経験を共有することもできます。
