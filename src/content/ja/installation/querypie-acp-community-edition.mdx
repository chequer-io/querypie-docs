# QueryPie ACP Community Edition

### 🧭 始める前に

**QueryPie Community Edition**は**QueryPie Enterprise**で提供される核心機能を体験できるようにします。

* Database Access Controller
* System Access Controller
* Kubernetes Access Controller
* Web Access Controller

ただし、Community Editionは**最大5名の利用者アクティベーション**をサポートします。*（5名以上を登録する必要がある場合、*[*Enterprise Plan*](https://www.querypie.com/ko/plans)*アップグレードを検討してください。）*

QueryPieは一般的なウェブアプリケーション形式で実行され、プロキシベースネットワークサーバ機能も含んでいます。

### 🐳 インストール方法

#### 1. ターミナルでコマンド実行

Linuxサーバのターミナルに接続した後、ホームディレクトリで次のコマンドを実行してください：
```
bash <(curl -s https://dl.querypie.com/setup.v2.sh)
```

⏱️ インストールは通常**7～10分**程度かかります。

#### 2. インストール中にライセンス発行を受ける

QueryPie Community Editionはライセンス登録が必要です。
インストールが進行中に[ライセンス申請書を提出](https://querypie.com/querypie/license/community/apply)すると、申請時に入力したメールアドレスに`.crt`テキストファイルが送信されます。

#### 3. インストール後接続

インストールが完了するとブラウザで次のアドレスに接続してください：
```
http://<server IP address>  
or 
https://<server IP address> 
```

QueryPieがインストールされたLinuxサーバのIPアドレスは利用者PCからアクセス可能である必要があります。

#### 4. ライセンス登録

メールで受け取った`.crt`ファイルをアップロードするか、PEM形式のテキスト内容をコピーして貼り付けてください。

新しいパスワードを設定した後サービスを使用できます。

* ID: `qp-admin`
* Password: `querypie`

### 🔑 ライセンス案内

* QueryPieを使用するにはインストール後必ず有効なライセンスを登録する必要があります。
* ライセンスは`.crt`拡張子のテキストファイルで提供されます。
* ライセンスは申請時に入力したメールアドレスに送信されます。
* 各ライセンスは発行日から1年間有効です。
* ライセンスは申請者本人に発行され、第三者に譲渡できません。

### 💬 サポートおよび問い合わせ

QueryPie Communityインストールおよび使用中に疑問点があれば[QueryPie AIP](https://app.querypie.com/)を通じて直接問い合わせできます。
以下の手順に従ってMCPで**QueryPie顧客センター**プリセットを設定すると、チャット画面で直接サポートを受けられます。

<figure data-layout="center" data-align="center">
![image-20250718-070933.png](/installation/querypie-acp-community-edition/image-20250718-070933.png)
</figure>

1. Integrationタブ → Install QueryPie Customer Centerクリック。
2. Presetタブ → New Preset選択後、以下のように入力して`Save Changes`クリック：

* Name: `QueryPie`を入力（推奨）。
* MCP Servers & Tools: `Select All`を選択。

1. Chatタブ → `@QueryPie`選択後、質問入力。

また、[公式QueryPie Discordチャネル](https://discord.gg/NsP98BHBm2)に参加して他の利用者と質問と経験を共有することもできます。
