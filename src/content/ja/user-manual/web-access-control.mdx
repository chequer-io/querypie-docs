# Web Access Control

### Overview

QueryPie WACはウェブアプリケーションに対するアクセスを制御しモニタリングするソリューションです。
社内管理者サイトとSaaSプラットフォームを含むすべてのウェブアプリケーションに対して役割基盤アクセス制御(RBAC)を適用してセキュリティを強化します。
ユーザーのすべてのウェブ活動をリアルタイムで追跡し記録して完全な可視性を提供します。
