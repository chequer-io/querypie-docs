# Kubernetes Access Control

### Overview

QueryPieはKubernetesクラスターに対するアクセス権限を制御し、Kubernetesクラスターに接続したユーザーのActivityを監査ログで記録する機能を提供します。
クエリパイエージェントを通じて権限を付与されたクラスターに接続でき、クラスター接続に必要なKUBECONFIGファイルはエージェントを通じて構成されます。
