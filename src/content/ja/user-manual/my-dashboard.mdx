# My Dashboard

### Overview

QueryPie Webログインが成功するとユーザーダッシュボード画面に移動します。
ダッシュボードでは現在ログインしたユーザーに付与された役割権限、所属グループなどの基本情報および決裁現況、最近接続したコネクション目録などを一目で確認できます。

### QueryPie Webにログイン

ウェブブラウザを通じてQueryPieにアクセスすると最も最初にログイン画面に遭遇します。

<Callout type="info">

</Callout>

#### Google Analyticsクッキー使用告知およびクッキー設定

クエリパイはユーザー体験向上のためにクッキーを使用して情報を収集します。
初回ログインページにアクセスした時選択的分析用クッキー使用に対する同意や拒否をすることができ同意または拒否はいつでもユーザーが修正できます。
選択的分析用クッキーに対する個別的な同意または拒否はクッキー設定(Cookie Preferences)画面で可能です。
必須クッキーは使用しない場合基本的な機能が動作しない場合があるため使用拒否が不可能です。
ログイン後にもpreferenceのユーザー環境設定画面のCookie & Privacy画面で設定できます。
インターネット通信が不可能な閉鎖網の場合は同意しても外部に情報を送信しません。
個人情報処理指針(Privacy Policy)リンクを通じてクエリパイのユーザー個人情報に対する取扱いポリシーを確認できます。<br/>

#### アカウント情報入力

*  **SSOログインボタン**  : 管理者がSSO(Single Sign-On)設定をした場合露出され、クリック時IdP認証ページに接続されます。
*  **ID/PW入力欄**  : QueryPieに登録されたID、パスワードを入力後`Continue`ボタンを押してログインを進行します。

### ユーザーダッシュボード

ログインが正常に完了されるとユーザーダッシュボード画面に移動します。
ダッシュボードでは現在ログインしたユーザーに付与された役割権限、所属グループなどの基本情報および決裁現況、最近接続したコネクション目録などを一目で確認できます。

#### ユーザー情報

*  **基本情報**  : ユーザー名(Display Name)、メール、最後のログイン時点が表示されます。
*  **Groups**  : ユーザーが属するユーザーグループが表示されます。
*  **Roles**  : ユーザーに割り当てられた役割目録が製品領域別に区分されて表示されます。管理者(Admin)役割が付与されたユーザーの場合、割り当てられたAdmin Roleが一緒に表示されます。

#### Workflow Requests

*  **Sent**   **Requests**  : 私が送った決裁要求中、進行中の要求数が表示されます。
*  **Received**   **Requests**  : 私が受けた要求中、私の承認が必要な要求数が表示されます。
*  **Submit Request**  : 該当ボタンを押してダッシュボードからすぐに決裁要求作成ページにアクセスできます。

#### Services

* 現在保有しているライセンスに応じて、使用できるサービスが表示されクリック時各サービスのメイン画面に移動します。

#### Recent Connections

* QueryPieサービス別で最近接続履歴があるコネクション目録を確認できます。
