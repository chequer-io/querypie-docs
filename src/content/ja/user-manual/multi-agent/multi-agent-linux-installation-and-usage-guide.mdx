# Multi Agent Linux インストールおよび使用ガイド

<Callout type="info">
QueryPie Multi AgentはFreeDesktop環境のLinuxプラットフォームでのみ動作します。
</Callout>

## サポート対象

|  **ベース**  |  **ディストリビューション**  |  **最小サポートバージョン**      |  **検証済みバージョン（開発チームで検証）**  |
| -------- | --------- | ------------------ | ----------------------- |
| Debian   | Debian    | 10 (Buster)        | 12 (Bookworm)           |
| Debian   | Ubuntu    | 18.04 LTS (Bionic) | 24.04 LTS (Noble)       |
| Debian   | Kali      | 2018.4             | 2024.2                  |
| RedHat   | Fedora    | 28                 | 40                      |
| RedHat   | RHEL      | 8 (Ootpa)          | -                       |
| RedHat   | CentOS    | 8.0                | -                       |

*  *最小サポートバージョン: glibc 2.27をサポートする最小Linuxバージョン* 

## インストールおよび削除方法

### インストールファイルのダウンロード

QueryPie Web &gt; プロフィール &gt; Multi-Agentを通じてMulti Agentインストールファイルをダウンロードできます。

### 事前作業

Multi Agentは[StatusNotifierItem](https://www.freedesktop.org/wiki/Specifications/StatusNotifierItem/)でトレイアイコン機能を提供します。

様々なLinuxディストリビューションでStatusNotifierItemがデフォルトで表示されない場合があります。

#### 1. インストール確認

```
gnome-extensions info appindicatorsupport@rgcjonas.gmail.com
```

**インストールされていない**
```
Extension “appindicatorsupport@rgcjonas.gmail.com” doesn't exist
```

**インストールされている**

<Callout type="important">
State: ENABLEDであることを必ず確認する必要があります。
</Callout>
```
appindicatorsupport@rgcjonas.gmail.com
  Name: AppIndicator and KStatusNotifierItem Support
  Description: Adds AppIndicator, KStatusNotifierItem and legacy Tray icons support to the Shell
  Path: /home/parallels/.local/share/gnome-shell/extensions/appindicatorsupport@rgcjonas.gmail.com
  URL: https://github.com/ubuntu/gnome-shell-extension-appindicator
  Version: 53
  State: ENABLED
```

#### 2. インストール

以下のリンクでAppIndicator and KStatusNotifierItem Supportをダウンロードし、インストールしてください。

[https://extensions.gnome.org/extension/615/appindicator-support/](https://extensions.gnome.org/extension/615/appindicator-support/)

### Debianベース

dpkgコマンドを基準に説明します。

#### Install

1. `querypie-multi-agent.deb`をダウンロードします。
2. 以下のコマンドを実行してファイルをインストールします。 <br/> 出力例
  ```
  sudo dpkg -i ~/Downloads/querypie-multi-agent.deb
  ```
  ```
  Preparing to unpack querypie-multi-agent.deb ...
  Unpacking querypie-multi-agent (3) ...
  Setting up querypie-multi-agent (3) ...
  Processing triggers for gnome-menus (3.36.0-1.1ubuntu3) ...
  Processing triggers for desktop-file-utils (0.27-2build1) ...
  Processing triggers for hicolor-icon-theme (0.17-2) ...
  ```
3. インストール状況を確認します。 <br/> 出力例
  ```
  dpkg -s querypie-multi-agent
  ```
  ```
  Package: querypie-multi-agent
  Status: install ok installed
  Priority: optional
  Section: devel
  Maintainer: QueryPie
  Architecture: arm64
  Version: 3
  Depends: libx11-6, libice6, libsm6, libfontconfig1
  Description: QueryPie Multi Agent
  Homepage: https://www.querypie.com/
  Copyright: QueryPie, Inc.
  ```

#### Uninstall

ターミナルで以下のコマンドを実行します。
```
sudo dpkg -r querypie-multi-agent
```

出力例
```
Removing querypie-multi-agent (3) ...
Processing triggers for hicolor-icon-theme (0.17-2) ...
Processing triggers for gnome-menus (3.36.0-1.1ubuntu3) ...
Processing triggers for desktop-file-utils (0.27-2build1) ...
```

### RedHatベース

rpmコマンドを基準に説明します。

#### Install

1. `querypie-multi-agent.rpm`をダウンロードします。
2. ターミナルで以下のコマンドを実行してファイルをインストールします。 <br/> 出力例
  ```
  sudo rpm -i ~/Downloads/querypie-multi-agent.rpm
  ```
  ```
  QueryPie Multi Agent has been installed.
  ```
3. インストール状況を確認します。 <br/> 出力例
  ```
  rpm -qi querypie-multi-agent
  ```
  ```
  Name        : querypie-multi-agent
  Version     : 3
  Release     : 202503291846
  Architecture: aarch64
  Install Date: Mon 31 Mar 2025 03:02:34 PM KST
  Group       : Applications/Development
  Size        : 493101340
  License     : Proprietary
  Signature   : (none)
  Source RPM  : querypie-multi-agent-3-202503291846.src.rpm
  Build Date  : Sun 30 Mar 2025 03:46:29 AM KST
  Build Host  : localhost
  Vendor      : QueryPie, Inc.
  URL         : https://www.querypie.com/
  Summary     : QueryPie Multi Agent
  Description :
  QueryPie Multi Agent
  ```

#### Uninstall

ターミナルで以下のコマンドを実行します。
```
sudo rpm -e querypie-multi-agent
```

出力例
```
QueryPie Multi Agent is being uninstalled.
QueryPie Multi Agent has been uninstalled.
```


### Keyring

<Callout type="important">
Ubuntuを基準に説明します。
</Callout>

GNOMEデスクトップ環境では、ユーザーのパスワード、暗号化キー、証明書などを安全に保存し管理するシステムがあります。

Passwords and Keysアプリで**Loginストレージがロックされていると**マルチエージェント実行時に管理者権限を要求します。

## Troubleshooting

### Multi Agent内のリンクをクリックしましたが、反応がありません。

Default Browserを設定しないとリンクが開きません。
ブラウザ内の設定でデフォルトブラウザを設定してください。
