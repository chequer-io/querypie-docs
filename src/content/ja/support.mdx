# サポート

### 共通リソース

すべてのQueryPieのお客様がご利用いただける資料です。

#### Documentation

このウェブサイト、[docs.querypie.com](http://docs.querypie.com)では、以下の資料をご覧いただけます：

* ユーザーマニュアル
* 管理者マニュアル
* API Reference
* リリースノート

技術サポートのための[Confluence Space](https://querypie.atlassian.net/wiki/spaces/QCP/overview)で詳細な技術サポート資料をご確認いただけます。

* [QueryPie Architecture](https://querypie.atlassian.net/wiki/spaces/QCP/pages/400064797)
* [Advanced Environment Setup](https://querypie.atlassian.net/wiki/spaces/QCP/pages/887947577)
* [Advanced Integration Guide](https://querypie.atlassian.net/wiki/spaces/QCP/pages/841449834)
* [リリースバージョン別ドキュメント](https://querypie.atlassian.net/wiki/spaces/QCP/pages/841351486)
* [Troubleshooting](https://querypie.atlassian.net/wiki/spaces/QCP/pages/920486841)

#### Product Demo - YouTube

[Product Demo - YouTube](https://www.youtube.com/playlist?list=PLB6l6itD4YrpMXRtprvUdR3hIHVbT8IhZ)チャンネルで、QueryPie ACP製品のさまざまな機能を動画でご覧いただけます。

* DAC：データベースアクセス制御機能のチュートリアル
* SAC：サーバーアクセス制御機能のチュートリアル
* KAC：Kubernetesアクセス制御機能のチュートリアル
* General：SSO統合、監査ログ、BIツール連携（Tableau、Redash）
