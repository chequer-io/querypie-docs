# 管理者マニュアル

### ようこそ。

QueryPieの管理者は、QueryPie WebまたはAPIを通じて、リソースおよび権限、ソリューションセキュリティ、通知設定など、QueryPie全般のソリューション管理を進めることができます。
提供されたライセンスおよび管理者役割権限に応じて、管理者に表示される設定メニューが異なります。

管理者ページにアクセスするには、まずQueryPieにログインします。
ログインが正常に完了すると、ユーザーダッシュボードに移動し、QueryPie管理者役割が割り当てられたユーザーの場合、右上に`Go to Admin Page`ボタンが表示されます。
このボタンをクリックして管理者ページを新しいタブで開くことができます。

### さあ、基本設定が完了しました。今度はサービス別設定をしてみましょうか？

基本設定が完了したら、サービス別設定が残っています。
QueryPieの細分化されたアクセス制御機能を通じて、強力な権限管理を体験してみてください。
サービス別設定が終わった後は、管理者/ユーザーの接続および実行履歴をAuditメニューで詳しく確認できます。
また、機密情報を識別できるDiscovery機能も提供するので、次世代ディスカバリ機能を体験してみてください。
