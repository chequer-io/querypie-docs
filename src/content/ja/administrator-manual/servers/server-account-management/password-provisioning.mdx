# Password Provisioning

### Overview

パスワードを定期的に変更するJobを生成し管理するメニューです。
サーバーグループ単位でJobを生成し、パスワード変更はサーバーグループに属する個別サーバーを対象にアカウント別プロビジョニングを実行します。

<Callout type="info">
このメニューはAdministrator &gt; Servers &gt; [SAC General Configurations](../sac-general-configurations)で  **Password Provisioning**  設定をOnに選択後保存すると表示されます。
</Callout>


### パスワード変更Job照会

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/servers/server-account-management/password-provisioning/Screenshot-2025-07-27-at-6.31.29-PM.png" alt="Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning" width="760" />
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning
</figcaption>
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioningメニューに移動します。
2. テーブル左上の検索欄を通じてJobのNameまたはProvisioning Accountの検索が可能です。
3. テーブル右上のリフレッシュボタンを通じてアカウントリストを最新化できます。
4. テーブルで以下のカラム情報を提供します。
    1.  **Name** : パスワード変更JobのName
    2.  **Provisioning Account**  : 実際のサーバーに接続してパスワード変更を実行するAccount
    3.  **Job Frequency**  : Job実行周期
    4.  **Latest Result**  : 最後の実行結果
    5.  **Last Run At**  : 最後の実行日時
5. 各行をクリックするとJobの詳細情報照会が可能です。

2. 画面右上の`Run Job`ボタンをクリックします。

### パスワード変更Job生成

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioningメニューに移動します。
2. 右上の`+ Create Job`ボタンをクリックします。
3. [パスワード変更Job生成](password-provisioning/creating-password-change-job)ガイドを参照してサーバーを登録します。


### パスワード変更Job削除

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/servers/server-account-management/password-provisioning/image-20240831-161802.png" alt="image-20240831-161802.png" width="414" />
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioningメニューに移動します。
2. テーブル内削除対象Job左側のチェックボックスをチェックします。
3. テーブルカラムラインに現れた`Delete`ボタンをクリックします。
4. ポップアップが現れたら`OK`ボタンをクリックして削除を進行します。
