# Server Account Management

### Overview

この文書では、QueryPieでサーバーアカウントを効果的に管理するためのガイドを確認できます。
サーバーアカウント登録のためのテンプレートやSSHキー管理機能も提供され、サーバーのアカウントに対するパスワード更新自動化を設定できます。
