# KAC General Configurations

<Callout type="info">
10.3.0から各Administrator &gt; General &gt; Security下位にあったKAC関連設定項目たちが各サービスメニューのGeneral &gt; Configurationsに移動されました。
11.4.0からユーザーの権限要求および管理者の権限付与時最大期間を指定できるMaximum Access Durationオプションが追加されました。
</Callout>


### Kubernetesアクセス統制設定

Kubernetesクラスターアクセス制御、権限管理、セッションポリシーなどコンテナベース環境でのセキュリティのための設定を提供します。

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/kubernetes/kac-general-configurations/image-20251107-090909.png" alt="Kubernetes Configurations options" width="760" />
<figcaption>
Kubernetes Configurations options
</figcaption>
</figure>


* Idle Session Timeout (Minutes) : ポッドセッションタイムアウト (Default : 15)
    * exec、port-forward、attachを通じたセッション接続時アイドル時間指定
* Maximum Access Duration : 権限付与最大期間指定オプションです。指定した期間を超過する権限要求を作成できず、管理者もこの期間を超過した権限付与をできません。この期間が満了すると自動で権限が回収されます。 
    * 最小値 : 1、最大値 : 9999
