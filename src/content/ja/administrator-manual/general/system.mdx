# System

### Overview

QueryPieのシステム設定を管理するメニューです。
各下位メニューで外部セキュリティサービスとの連動、External API Token、QueryPie内部で定期的に実行される作業を管理できます。

### 下位メニュー

それぞれのメニューは以下のような機能を提供します。

* [Integrations](system/integrations) ：SIEMサービスおよびSecret Store連動管理
* [API Token](system/api-token) ：External API Token管理
* [Jobs](system/jobs) ：QueryPie内部の定期的/手動作業一覧

詳細内容は各下位メニューに対する説明を参照してください。
