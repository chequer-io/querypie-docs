# Web App Access Control

### Overview

この文書では、QueryPieでウェブアプリアクセス権限を効果的に管理するためのガイドを確認できます。
RBACを通じて事前に定義したアクセス権限を繰り返し付与できる方法を提供しており、URLベースのウェブアプリアクセス制御を通じてユーザー別に許可されたウェブアプリケーションにのみアクセスできるように管理します。
また、ポリシー管理機能を通じて詳細なアクセス規則を設定できます。
