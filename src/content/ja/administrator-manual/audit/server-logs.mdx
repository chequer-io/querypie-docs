# Server Logs

### Overview

QueryPieのSystem Access Controlで発生するログをServer Logsで確認できます。
サーバー接続権限付与と回収、サーバーログイン/ログアウト、実行したコマンド履歴などの情報を確認できます。
その他、QueryPieを通じて接続したセッションリストを照会したりセッションを強制終了できます。

1.  **General** 

* [User Access History](general-logs/user-access-history) : ユーザーログイン、ログアウト履歴
* [Activity Logs](general-logs/activity-logs) : 管理者が実行するリソース登録および設定変更履歴
* [Admin Role History](general-logs/admin-role-history) : 管理者権限付与、変更、回収履歴

1.  **Servers** 

* [Server Access History](server-logs/server-access-history) : Server接続履歴
* [Command Audit](server-logs/command-audit) : Server内実行コマンド履歴
* [Session Logs](server-logs/session-logs) : 接続セッション内行為録画履歴
* [Session Monitoring (Moved)](server-logs/session-monitoring-moved) : 接続したセッションリスト照会
* [Access Control Logs](server-logs/access-control-logs) : Direct Permission付与履歴
* [Server Role History](server-logs/server-role-history) : Role付与履歴
* [Account Lock History](server-logs/account-lock-history) : QueryPieユーザー別サーバー接続アカウントロックおよび解除履歴

<Callout type="info">
11.5.0からAudit配下メニュー項目であるSession MonitoringはServers &gt; Monitoring &gt; Session Monitoringに移動されました。
</Callout>
