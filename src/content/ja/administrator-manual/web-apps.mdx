### Overview

QueryPie WAC（Web App Access Control）は、ウェブアプリケーションに対するアクセス制御とログ記録のためのWeb Application保護ソリューションです。

WACで提供する詳細機能は以下の通りです。

* アクセス制御：許可されたユーザーのみにウェブアプリケーションへのアクセスを許可します。
* 監査ログ：許可されたユーザーであっても、該当ウェブアプリケーションにアクセスして作業した履歴を記録します。

また、機能更新を通じて以下のような機能を提供予定です。

* 権限制御：許可されたユーザーであっても、該当ウェブアプリケーションの主要作業に対してはその行為を制御します。
* 機密情報制御：許可されたユーザーに許可された作業であっても、重要情報に対しては表示を制限します。


### Web Apps下位機能

WAC管理メニューに進入するには、QueryPie Adminページ上部メニューでWeb Appsをクリックします。

下位メニュー別の簡略説明は以下の通りです。

* Connection Management
    * Web Apps：制御対象ウェブアプリを登録し管理します。
* Web App Access Control
    * Access Control：QueryPieグループとユーザーに役割を付与または回収します。
    * Roles：Web App役割を管理します。役割生成/修正/削除、役割別ユーザーと許可されたウェブアプリ照会、ポリシー割り当て/回収などが可能です。
    * Policies：Web Appポリシーを管理します。ウェブアプリに対するアクセス許可/ブロックなど詳細ポリシーを生成でき、該当ポリシーが割り当てられた役割を照会し、ポリシーの修正履歴を確認できます。
