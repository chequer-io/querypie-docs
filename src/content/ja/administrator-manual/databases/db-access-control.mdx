# DB Access Control

### Overview

DB Access Controlメニューでは、高いレベルでデータベースアクセスおよびセキュリティを管理するために役割と属性を基にした多様な形態のアクセス制御を提供します。
認証されたユーザーに限ってQueryPieを通じてアクセス権限が付与されたデータベースにのみアクセスでき、SQL実行は事前に定義されたSQL構文組み合わせを基に実行が制限されます。
最初のQueryPieユーザー認証からデータベースアクセスまでIP帯域別統制、長期未接続統制、時間および曜日別統制、テーブル/カラム照会制限などで重要データを安全に保護でき、セキュリティポリシーに対するすべての変更事項はユーザーセッションに即座に適用されてアップデートされた措置をリアルタイムで適用できます。
