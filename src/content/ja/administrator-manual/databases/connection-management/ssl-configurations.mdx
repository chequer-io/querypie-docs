# SSL Configurations

### Overview

SSL ConfigurationsページでDBコネクションに接続する際に使用するSSL構成情報を登録します。

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/databases/connection-management/ssl-configurations/screenshot-20240730-230107.png" alt="Administrator &gt; Databases &gt; Connection Management &gt; SSL Configuration &gt; Create SSL" width="760" />
<figcaption>
Administrator &gt; Databases &gt; Connection Management &gt; SSL Configuration &gt; Create SSL
</figcaption>
</figure>


1. Databases設定メニューからConnection Management &gt; SSL Configurationメニューに移動します。
2. 右上の`Create SSL`ボタンをクリックします。
3. SSL証明書登録のための以下の情報を入力します。
    1. Name : SSL証明書を画面上で区別できる名前です。
    2. SSL Mode : SSL接続してアクセスする際の接続モードを選択します。
    3. Private Key : SSL接続後、データの暗号化/復号化に必要なキーを入力します。
    4. Certificate : クライアント情報を識別するための証明書情報を入力します。
    5. CA Certificate : 証明書の検証のためのCA証明書情報を入力します。
4. `Save`ボタンで保存します。
