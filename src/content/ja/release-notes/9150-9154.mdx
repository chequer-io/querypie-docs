### QueryPie 9.15.0 Release

2023年12月08日


**New Features**

* [Security] パスワード期限サイクル設定およびリセット機能を追加
* [Workflow] バックグラウンドクエリ実行
* [Audit] Log Exportメニューを新規追加
* [DAC] Secret Store連携機能追加およびDB認証情報へのVault連携をサポート
* [DAC] Snowflake接続をサポート


**Enhancement**

* [Web Editor] Preview処理UXを改善
* [IDP] LDAP同期時の2個以上グループ連携をサポート
* [Agent] エージェント内にQueryPie Host選択リストを追加
* [Agent] エージェントログインタイムアウト更新ロジックを改善
* [DAC] Redis SSL/TLSをサポート
* [SAC] SSH証明書追加対応


**Bug Fix**

* [Athena] AwsDataCatalogでデータベースを変更できない
* [MongoDB] Import時の「"undefined" is not valid JSON」というエラーが発生
* [Workflow] SQL Requestファイルタイプリクエスト再上申時にエディターモードを使用できない問題
* [Editor] Querypie Export、mysqldump問題を解決
* [Agent] 接続リストのExposeオプション処理が反映されるよう修正
* [Athena] IAM権限によるエラー発生時のDB接続切断問題を改善
