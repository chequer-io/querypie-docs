### Overview

QueryPie WAC(Web App Access Control)은 웹 애플리케이션에 대한 접근통제 및 로깅을 위한 Web Application 보호 솔루션입니다.

WAC에서 제공하는 세부 기능은 다음과 같습니다.

* 접근 통제 : 허가된 사용자에게만 웹 애플리케이션에 대한 접근을 허용합니다.
* 감사 로그 : 허가된 사용자라 하더라도 해당 웹애플리케이션에 접근하여 작업한 이력을 기록합니다.

또한 기능 업데이트를 통하여 다음과 같은 기능을 제공할 예정입니다.

* 권한 통제 : 허가된 사용자라 하더라도 해당 웹애플리케이션의 주요 작업에 대해서는 그 행위를 통제합니다.
* 민감정보 통제 : 허가된 사용자에게 허가된 작업이라 하더라도 중요 정보에 대해서는 표시를 제한합니다.


### Web Apps 하위 기능

WAC 관리 메뉴에 진입하려면 QueryPie Admin 페이지 상단 메뉴에서 Web Apps를 클릭합니다.

하위 메뉴별 간략한 설명은 다음과 같습니다.

* Connection Management
    * Web Apps : 통제 대상 웹 앱을 등록하고 관리합니다.
* Web App Access Control
    * Access Control : QueryPie 그룹 및 사용자에게 역할을 부여 또는 회수합니다.
    * Roles : Web App 역할을 관리합니다. 역할 생성/수정/삭제, 역할별 사용자 및 허용된 웹 앱 조회, 정책 할당/회수 등이 가능합니다.
    * Policies : Web App 정책을 관리합니다. 웹 앱에 대한 접근 허용/차단 등 세부 정책을 생성할 수 있고, 해당 정책이 할당된 역할을 조회하고, 정책의 수정 내역을 확인할 수 있습니다.
