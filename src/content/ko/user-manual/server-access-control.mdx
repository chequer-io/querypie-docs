# Server Access Control

### Overview

QueryPie 에서는 통제된 환경에서 접속 권한이 있는 서버에 대해서만 접속할 수 있습니다.
커넥션에 대해 접속 권한을 부여받은 사용자는 웹 터미널 또는 웹 SFTP를 통해 커넥션에 접속하여 여러 명령을 실행할 수 있습니다.
접속 권한은 개별 사용자에게 부여될 수도 있고, 사용자 그룹에 부여될 수도 있습니다.
