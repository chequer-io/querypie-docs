# 11.4.0

### QueryPie 11.4.0 Release

2025년 10월 31일


**New Feature**

* [공통] 대리결재자 지정 이벤트에 대한 감사 로그 기록 
* [DAC] Web Editor에서 쿼리의 크기 표시
* [DAC] ShardingSphere 지원
* [KAC] Configurations에 Max Access Duration 설정 추가


**Improvement**

* [공통] 관리자 Users 메뉴에서 기본 검색 조건 필터를 Name으로 변경
* [공통] 에이전트에 등록 할 3rd party Tool 등록 실패시 에러 메시지 개선
* [공통] Alert 메뉴에서 알림 항목 생성 후 채널 변경 못하도록 변경
* [공통] Alert 메뉴에서 Channel을 Email로 선택 시, Subject 필드가 표기되도록 개선
* [공통] Audit Log Export Log Type 정렬 개선
* [공통] Event Callback 설정 시 과부하 상황이 아니라면 이벤트 발생 시 즉시 수신하도록 개선
* [공통] Reverse Tunnel 메뉴 위치 변경
* [공통] Workflow의 “Use User Attribute-Based Approval” 에 연결되는 LDAP Attribute 복수 개 지정 할 수 있도록 개선
* [DAC] 오라클 프로시저 목록 조회 시 ORA-19011 / ORA-01489 에러 발생 이슈 수정
* [DAC] DB Access control External API 호출시 granted at, expiration date 를 결과에 포함하도록 개선
* [DAC] Web Editor에서 Impala / Hive 의 Data Tab 비활성화
* [DAC] MySQL 비밀번호 만료시 사용자가 본인 DB 계정의 비밀번호를 변경할 수 있도록 개선
* [DAC] Trino에 대해 DML 옵션 비활성화로 변경
* [DAC] New DAC Policy Management 기능에 Unmasking Zone 적용
* [DAC] New DAC Policy Management 기능의 policy audit log를 스냅샷 형태로 개선
* [KAC] Allowed Kubernetes Users 필드 placeholder 수정
* [KAC] 다중 Role이 부여된 사용자가 접속시 Role 선택하는 기능 제거
* [SAC] Direct Permission에서 Allow 형태의 Command Template 지원
* [SAC] Multi-Agent에서 서버 그룹 목록 조회 개선
* [SAC] ProxyJump 생성 시 Jump host의 auth type을 수정하더라도 Account 입력 정보를 유지하도록 개선
* [SAC] Server Access/Privilege Request를 Minute-Based로 요청(상신) 시, Expiration Date 값은 자동으로 계산된 값이 보이도록 비활성화 상태로 표기
* [SAC] YAML로 SAC Policy 입력 시, Protocol 항목의 값을 대소문자 구분없이 입력 가능하도록 개선
* [WAC] Web Secure 다운로드 화면에서 새로운 설치가이드 페이지로 링크 수정
