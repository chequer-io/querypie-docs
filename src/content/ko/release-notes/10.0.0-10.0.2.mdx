### QueryPie 10.0.0 Release

2024년 07월 22일


**New Features**

* [General] 사용자/관리자 페이지 분리 제공 
* [General] 사용자 대시보드(My Dashboard) 메뉴 신설 
* [Security] 패스워드 재사용 금지 횟수 설정 기능 제공
* [AIDD] AI Data Discovery 기능 신규 출시 
* [Workflow] Access Role Request 신규 제공 
* [DAC] Connection 페이지 내 Tag 기능 제공
* [DAC] GCP 동기화 재지원 
* [DAC] Hive 커버로스 인증 지원 
* [DAC] SQLServer SSMS Proxy 접속 지원
* [SAC] 윈도우 서버 접근 제어(RDP) 지원


**Enhancement**

* [SCIM] 사용자 및 그룹 Import 기능 추가 및 Group Push 기능 개선 
* [Workflow] SQL Request 에서 쿼리 실행 후 중지(Stop) 할 수 있는 기능 제공
* [Audit] Audit Log Export 목록에서 Task별 상태값 확인 가능하도록 개선 
* [Audit] Audit Log Export 에서 Workflow SQL Request 타입 추출 시 ID 값 추가 제공
* [DAC] SQL Editor로 여러 커넥션 연결 시 세션을 일괄 종료할 수 있는 기능 제공


**Bug Fix**

* [SCIM] Profile Editor에 따른 Attribute 업데이트 비정상 동작 이슈 해결
* [External API] 9.19.0 업그레이드 후 API Token 권한 Scopes 값이 Select Role 로 부여되는 문제 해결
* [DAC] SQL Server 특정 쿼리(IF/EXEC)가 실행되지 않는 문제 수정
* [DAC] 브라우저 신규 탭으로 Connection 접속 시 간헐적 오류 발생하는 문제 수정
* [Activity Log] null 값이 포함된 과거 데이터 조회 시 에러 발생하는 문제 해결 
* [Security] QueryPie Web IP Access Control 내 특정 값 설정 시 에러 발생하는 문제 해결
* [Agent] MariaDB 커넥션을 Open Connection With 기능으로 연결 시 에러 발생하는 문제 해결 
* [DAC] MariaDB Temporary Table의 DMLSnapshot 값이 On인 경우 정상 동작 안하는 버그 수정
* [Agent] Ubuntu에서 Firefox로 Agent 접속 불가한 문제 해결


**External API**

* Workflow - Access Role Request Details 조회 API 추가
