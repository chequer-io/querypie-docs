# QueryPie ACP Community Edition

### 🧭 시작하기 전에

**QueryPie Community Edition** 은  **QueryPie Enterprise** 에서 제공되는 핵심 기능을 체험할 수 있도록 합니다.

* Database Access Controller
* System Access Controller
* Kubernetes Access Controller
* Web Access Controller

단, Community Edition은  **최대 5명의 사용자 활성화** 를 지원합니다.  *(5명 이상을 등록해야 하는 경우,* [ *Enterprise Plan* ](https://www.querypie.com/ko/plans) *업그레이드를 고려해 주세요.)*

QueryPie는 일반적인 웹 애플리케이션 형태로 실행되며, 프록시 기반 네트워크 서버 기능도 포함하고 있습니다.

### 🐳 설치 방법

#### 1. 터미널에서 명령 실행

Linux 서버의 터미널에 접속한 후, 홈 디렉터리에서 다음 명령어를 실행하세요:
```
bash <(curl -s https://dl.querypie.com/setup.v2.sh)
```

⏱️ 설치는 보통  **7~10분**  정도 소요됩니다.

#### 2. 설치 중 라이선스 발급 받기

QueryPie Community Edition은 라이선스 등록이 필요합니다.
설치가 진행되는 동안 [라이선스 신청서를 제출](https://querypie.com/querypie/license/community/apply)하면, 신청 시 입력한 이메일 주소로 `.crt` 텍스트 파일이 발송됩니다.

#### 3. 설치 후 접속

설치가 완료되면 브라우저에서 다음 주소로 접속하세요:
```
http://<server IP address>  
or 
https://<server IP address> 
```

QueryPie가 설치된 Linux 서버의 IP 주소는 사용자 PC에서 접근 가능해야 합니다.

#### 4. 라이선스 등록

이메일로 받은 `.crt` 파일을 업로드하거나, PEM 형식의 텍스트 내용을 복사해 붙여넣으세요.

### 🔑 라이선스 안내

* QueryPie를 사용하려면 설치 후 반드시 유효한 라이선스를 등록해야 합니다.
* 라이선스는 `.crt` 확장자의 텍스트 파일로 제공됩니다.
* 라이선스는 신청 시 입력한 이메일 주소로 발송됩니다.
* 각 라이선스는 발급일로부터 1년간 유효합니다.
* 라이선스는 신청자 본인에게 발급되며, 제3자에게 양도할 수 없습니다.

### 💬 지원 및 문의

QueryPie Community 설치 및 사용 중 궁금한 점이 있으면 [QueryPie AIP](https://app.querypie.com/)를 통해 직접 문의할 수 있습니다.
아래 절차에 따라 MCP에서  **QueryPie 고객센터**  프리셋을 설정하면, 채팅창에서 바로 지원을 받을 수 있습니다.

<figure data-layout="center" data-align="center">
![image-20250718-070933.png](/installation/querypie-acp-community-edition/image-20250718-070933.png)
</figure>

1. Integration 탭 → Install QueryPie Customer Center 클릭.
2. Preset 탭 → New Preset 선택 후 아래와 같이 입력하고 `Save Changes` 클릭:

* Name: `QueryPie` 를 입력(권장).
* MCP Servers & Tools: `Select All` 를 선택.

1. Chat 탭 → `@QueryPie` 선택 후 질문 입력.

또한, [공식 QueryPie Discord 채널](https://discord.gg/NsP98BHBm2)에 참여하여 다른 사용자들과 질문과 경험을 공유할 수도 있습니다.
