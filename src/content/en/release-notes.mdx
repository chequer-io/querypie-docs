# Release Notes

<Callout type="info">
You can check official QueryPie release information by version.

### Release Notes

*  **New Features** : Introduces features newly provided in the version. 
*  **Enhancement**  : Describes improvements to performance, usability, or convenience of existing features.
*  **Bug Fix**  : Lists issues and defects reported by customers that have been resolved. 
*  **External API**  : Announces newly provided APIs or changes to existing APIs.
