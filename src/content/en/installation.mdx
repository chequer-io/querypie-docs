# Product Installation

QueryPie ACP provides pre-built Docker images for all versions and can be easily and quickly installed on Linux VMs in Public Cloud or on-premise Linux server environments.
For installation instance specifications, we recommend 4 vCPUs, 16 GiB Memory, and at least 100 GiB of disk space.
