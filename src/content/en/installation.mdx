# Product Installation and Technical Support

### QueryPie Installation

QueryPie provides pre-built Docker images for all versions and supports easy and fast installation on EC2.
For installation instance specifications, we recommend 4 vCPUs, 16 GiB Memory, and at least 100 GiB of disk space.

For detailed installation instructions, please refer to the following documents:

* [Prerequisites](installation/prerequisites)
* [Installation](installation/installation)
* [System Architecture and Network Access Control](installation/system-architecture-and-network-access-control)
* [Container Environment Variables](installation/container-environment-variables)

### Upgrades and Routine Maintenance

When new versions of QueryPie are released, we notify you of the latest updates in the [Release Notes](release-notes).
We provide upgrade support for major version releases based on your annual subscription contract.
Minor version releases are conducted only for bug fixes.
For inquiries about upgrades and routine maintenance, please contact us through [QueryPie Customer Portal](https://help.support.querypie.com/).
