# User Profile

### Overview

Attribute-based Access Control (ABAC) can be implemented using attributes registered in the user profile.
This document provides detailed information for each user profile field.


### Profile 

This tab displays user attributes.
For descriptions of each attribute, please refer to the **Reference: User Attribute List** section at the bottom of this page.

<Callout type="important">
**Prerequisites**
To assign Allowed Zones per user, you must first enable the per-user QueryPie Web access IP control feature in the Admin &gt; General &gt; Company Management &gt; Security menu. Please refer to the [Security](../../company-management/security) documentation.
</Callout>

#### Viewing Static IPs Registered Through Workflow

When users request IP address registration through the 'IP Registration Request' workflow and receive final approval, the IP is automatically added to the `Static IP` attribute in the user's **Profile** tab.

These addresses registered in the `Static IP` attribute are automatically reflected in this `Allowed Zone` tab as items with `Name` set to `Static IP`, granting users access permissions from those IPs.

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/general/user-management/users/user-profile/Screenshot-2025-06-26-at-6.12.22-PM.png" alt="Screenshot-2025-06-26-at-6.12.22-PM.png" width="779" />
</figure>

#### Deleting Static IPs Registered Through Workflow

`Static IP` items automatically registered through workflow cannot be directly detached from the `Allowed Zones` tab.
To delete them, navigate to the user's **Profile** tab and directly edit the `Static IP` attribute value to remove the desired IP address. When values are deleted from the `Static IP` attribute, those IPs are immediately removed from the `Allowed Zones` tab and access permissions are revoked.


#### Table Display Information

The information displayed in the table is as follows:

1.  **Name** : Allowed Zone name
2.  **IP Bands**  : Allowed Zone IP range

#### Assigning Allowed Zones to Users

Click the `Attach Allowed Zones` button on the right side of the Allowed Zones tab to display the Allowed Zone assignment modal.
Select the Allowed Zone you want to assign and click the `Attach` button to complete the assignment.

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/general/user-management/users/user-profile/screenshot-20240730-214403.png" alt="Attach Allowed Zone Modal" width="760" />

#### Detaching Allowed Zones from Users

Select the item you want to detach using the checkbox in the list at the bottom of the Allowed Zone tab, and the `Detach` button will appear.
Click the button and then click the `Detach` button in the confirmation modal to complete the detachment.
