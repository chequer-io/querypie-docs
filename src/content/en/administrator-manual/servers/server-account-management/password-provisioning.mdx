# Password Provisioning

### Overview

This is a menu for creating and managing Jobs that periodically change passwords.
Jobs are created by server group unit, and password changes perform account-specific provisioning for individual servers belonging to the server group.

<Callout type="info">
This menu is displayed only after selecting **Password Provisioning** setting to On and saving in Administrator &gt; Servers &gt; [SAC General Configurations](../sac-general-configurations).
</Callout>


### Viewing Password Change Jobs

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/servers/server-account-management/password-provisioning/Screenshot-2025-07-27-at-6.31.29-PM.png" alt="Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning" width="760" />
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning
</figcaption>
</figure>

1. Navigate to Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning menu.
2. You can search for Job Name or Provisioning Account through the search field in the top left of the table.
3. You can refresh the account list through the refresh button in the top right of the table.
4. The table provides the following column information.
    1.  **Name** : Password change Job Name
    2.  **Provisioning Account**  : Account that actually accesses the server and performs password changes
    3.  **Job Frequency**  : Job execution cycle
    4.  **Latest Result**  : Last execution result
    5.  **Last Run At**  : Last execution date and time
5. Clicking each row allows you to view Job detailed information.

2. Click the `Run Job` button in the top right of the screen. 

### Creating Password Change Jobs

1. Navigate to Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning menu.
2. Click the `+ Create Job` button in the top right.
3. Register servers by referring to the [Creating Password Change Jobs](password-provisioning/creating-password-change-job) guide.


### Deleting Password Change Jobs

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/servers/server-account-management/password-provisioning/image-20240831-161802.png" alt="image-20240831-161802.png" width="414" />
</figure>

1. Navigate to Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning menu.
2. Check the checkbox on the left of the target Job to delete in the table.
3. Click the `Delete` button that appears on the table column line.
4. When a popup appears, click the `OK` button to proceed with deletion.
