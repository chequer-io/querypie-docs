### Overview

QueryPie KAC is a Kubernetes API protection solution for centrally managing cloud infrastructure such as AWS EKS or other on-premises Kubernetes clusters. With KAC, administrators can perform detailed user Kubernetes resource API access restriction and management, and audit and monitor Kubernetes APIs requested by users.
