### Overview

QueryPie DAC can control 20+ types of RDBMS, NoSQL, and other diverse data sources through standardized SQL syntax using its own developed engine.
Users can access databases and execute queries through web browsers regardless of operating system.
It also supports the use of IDE tools such as Datagrip and DBeaver through desktop agents.
