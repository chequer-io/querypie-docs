# SSL Configurations

### Overview

Register SSL configuration information to use when connecting to DB connections on the SSL Configurations page.

<figure data-layout="center" data-align="center">
<img src="/administrator-manual/databases/connection-management/ssl-configurations/screenshot-20240730-230107.png" alt="Administrator &gt; Databases &gt; Connection Management &gt; SSL Configuration &gt; Create SSL" width="760" />
<figcaption>
Administrator &gt; Databases &gt; Connection Management &gt; SSL Configuration &gt; Create SSL
</figcaption>
</figure>


1. Navigate to Connection Management &gt; SSL Configuration menu from the Databases settings menu.
2. Click the `Create SSL` button in the top right.
3. Enter the following information for SSL certificate registration:
    1. Name : Name that can distinguish the SSL certificate on screen.
    2. SSL Mode : Select connection mode when connecting through SSL.
    3. Private Key : Enter the key required for data encryption/decryption after SSL connection.
    4. Certificate : Enter certificate information to identify client information.
    5. CA Certificate : Enter CA certificate information for certificate verification.
4. Save through the `Save` button.
