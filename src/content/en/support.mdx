# Support

### Common Resources

Resources available to all QueryPie customers.

#### Documentation

On this website, [docs.querypie.com](http://docs.querypie.com), you can find the following resources:

* User Manual
* Administrator Manual
* API Reference
* Release Notes

#### Product Demo - YouTube

Visit the [Product Demo - YouTube](https://www.youtube.com/playlist?list=PLB6l6itD4YrpMXRtprvUdR3hIHVbT8IhZ) channel to explore various features of the QueryPie ACP product through videos.

* DAC: Database Access Control feature tutorials
* SAC: Server Access Control feature tutorials
* KAC: Kubernetes Access Control feature tutorials
* General: SSO integration, audit logs, BI tool integration (Tableau, Redash)

### Community Support

For: Community Edition and Standard Edition customers

#### GitHub Discussion

If you have questions during installation or use, you can ask them on [GitHub Discussion](https://github.com/querypie/querypie-community/discussions) or share your experiences with other users.
