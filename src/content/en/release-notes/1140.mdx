# 11.4.0

### QueryPie 11.4.0 Release

October 31, 2025


**New Feature**

* [Common] Record audit log for delegate approver assignment events
* [DAC] Show query size in Web Editor
* [DAC] Support ShardingSphere
* [KAC] Add Max Access Duration setting to Configurations


**Improvement**

* [Common] Change default search filter in administrator Users menu to Name
* [Common] Improve error messages when 3rd party Tool registration fails for agent
* [Common] Prevent changing channel after creating alert item in Alert menu
* [Common] Show Subject field when selecting Email as Channel in Alert menu
* [Common] Improve Audit Log Export Log Type sorting
* [Common] Receive events immediately when Event Callback is configured, unless under overload conditions
* [Common] Change Reverse Tunnel menu location
* [Common] Allow specifying multiple LDAP Attributes connected to "Use User Attribute-Based Approval" in Workflow
* [DAC] Fix ORA-19011 / ORA-01489 errors when querying Oracle procedure list
* [DAC] Include granted at and expiration date in results when calling DB Access control External API
* [DAC] Disable Data Tab for Impala / Hive in Web Editor
* [DAC] Allow users to change their own DB account password when MySQL password expires
* [DAC] Change DML option to disabled for Trino
* [DAC] Apply Unmasking Zone to New DAC Policy Management feature
* [DAC] Improve policy audit log in New DAC Policy Management feature to snapshot format
* [KAC] Fix Allowed Kubernetes Users field placeholder
* [KAC] Remove feature to select Role when users with multiple roles connect
* [SAC] Support Allow type Command Template in Direct Permission
* [SAC] Improve server group list query in Multi-Agent
* [SAC] Maintain Account input information even when modifying auth type of Jump host when creating ProxyJump
* [SAC] Show Expiration Date value as disabled state with auto-calculated value when requesting (submitting) Server Access/Privilege Request as Minute-Based
* [SAC] Allow entering Protocol value case-insensitively when entering SAC Policy in YAML
* [WAC] Fix link to new installation guide page in Web Secure download screen
