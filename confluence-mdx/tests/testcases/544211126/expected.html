기능

1

로그인

대시보드 둘러보기

  • My Dashboard | 사용자-대시보드

Workflow로 권한 받기

  • DB Access Request 요청하기
  • SQL Request 요청하기

  • Unmasking Request 요청하기
  • Restricted Data Access 요청하기
  • Server Access Request 요청하기
  • Server Privilege Request 요청하기

  • IP Registration Request 요청하기
  • DB Policy Exception Request 요청하기

4

QueryPie Web에서 접속하기

  • Database Access Control
    • 웹 SQL 에디터로 접속하기
    • Default Privilege 설정하기
    • 에이전트 없이 프록시 접속하기
  • Server Access Control
    • 접근 권한이 있는 서버에 접속하기
    • 웹 터미널 사용하기
    • 웹 SFTP 사용하기
  • Kubernetes Access Control
    • 접근 권한 목록 확인하기
  • Web Access Control
    • Root CA 인증 서 및 Extension 설치하기
    • 웹 어플리케이션 (웹 사이트) 접속하기
